# 辞書機能をお試し

　たとえば、”　主人　”（しゅじん）　と入力してみてください。

　↓↓↓
　
　↑↑↑

　キャラクター設定に登録されているワードがサジェストされます。
　造語や、独自のキャラクター名でも、簡単に入力することができます。

　たとえば、”　ヒロ　”　と入力してみてください。
　↓↓↓
　
　↑↑↑

　辞書に登録されている単語をクリックすると、その定義を表示します。

　” 主人公は、ヒロインに目線を送った。 "



#  校正機能をお試し

　このエディターではJFTスタイルチェック、SudachiDict、Kuromoji.jsを使用しています。

　・” コンピュータ ”のような正しくない表記
　・" エディタ "のような文書内での表記ゆれ
　・” 饒舌 ”のような常用外の漢字

　などの校正を、外部のサーバーとは一切通信せず、デバイス上で全て処理し安全に使用することが出来ます、
