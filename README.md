## 書いてんかへようこそ！
　インストール不要で使える、Gitベースの物語執筆ツールです。  
　Chrome、Firefox、Safariなどのブラウザが動作する環境であればPC・スマートフォン問わずどこでも執筆ができます。  
  
　ブックマークに保存したり、ホーム画面に追加して、思い立ったときに書いてみてください。


## 広告掲載について
　広告・サブスクリプションは実装しません。
　皆様の応援だけが励みになります。


## ここからどうすればいいの？
　左側、ファイルツリーからチュートリアルファイルを開いて、色んな機能をお試ししてみてください。
　気に入ったら、設定タブからGitHub※1 にログインし、ご自身のリポジトリ※2 を作成して、執筆を開始できます。


## 上手く動かない、不具合を見つけた
　[こちら](https://portal.uo-uo.com/contact.html) からお問い合わせをお願いします。



※1 GitHub ... エンジニアが使う、データを安全に保存するためのウェブサイトです。
※2 リポジトリ ... GitHub上のあなただけが見れる個人フォルダ。書いてんかの設定画面から作成できます。
