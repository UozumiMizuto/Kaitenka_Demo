# 世界標準のデータ形式

　書いてんか のすべてのファイルは、

　.md
　.json

　といった世界標準のデータ形式で保存されます。　
　すべてのコンピューターで開くことのできるオープンなフォーマットです。




# 完全なデータ管理

　書いてんか で作成したデータは全て、世界中のエンジニアも使用している [GitHub](https://github.com/) に保存されます。
　たとえ 書いてんか がサービス終了したとしても、すべてのデータはあなたの手元に残ります。
　
　この機能デモも、[このように](https://github.com/UozumiMizuto/Kaitenka_Demo/blob/main/%E6%A9%9F%E8%83%BD%E7%B4%B9%E4%BB%8B/%E3%83%87%E3%83%BC%E3%82%BF%E4%BF%9D%E8%AD%B7.md) GitHubにすべてが保存されており、各データも簡単に確認できます。
